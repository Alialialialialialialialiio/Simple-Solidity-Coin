# Simple-Solidity-Coin



مشروع عملة رقمية بسيطة مبنية بلغة Solidity، تتيح إدارة الأرصدة والسيولة.

## الميزات (Features)
- **Minting**: إمكانية طباعة عملات جديدة (للمالك فقط).
- **Deposit**: تحويل الإيثيريوم إلى رصيد داخل العقد.
- **Withdraw**: سحب الإيثيريوم من العقد بناءً على الرصيد المتوفر.
- **ERC-20 Standards**: يتبع المعايير الأساسية لتسمية العملات (Name, Symbol, TotalSupply).

## المتطلبات التقنية
- Solidity ^0.8.0
- Remix IDE / Hardhat
